# project
# Описание проекта

Цель проекта — исследовать данные о покупках, чтобы выявить закономерности в поведении покупателей. Анализ включает изучение структуры данных, проверку на пропуски и дубликаты, поиск выбросов, описательную статистику, визуализацию и анализ корреляций.

# Структура проекта

shopping_behavior.csv — исходный датасет с данными о покупках.
Project_shopping_behavior.ipynb — основной файл с кодом анализа в формате Jupyter Notebook.
# Технологии

Проект написан на Python с использованием следующих библиотек:

pandas — для обработки данных
numpy — для численных операций
matplotlib и seaborn — для визуализации данных
scipy (неявно, для статистических тестов) — для поиска выбросов
# Основные шаги анализа

1. Загрузка и первичный осмотр данных

Загружен датасет shopping_behavior.csv.
Изучена структура данных: 3900 строк и 18 столбцов.
Просмотрены первые и последние 10 строк.
2. Проверка качества данных

Проверены типы данных и наличие пропусков — пропусков не обнаружено.
Проверены дубликаты — дубликатов нет.
Выполнен поиск выбросов с использованием Z-оценки — выбросы не обнаружены.
3. Описательная статистика

Для числовых столбцов (Age, Purchase Amount (USD), Review Rating, Previous Purchases) вычислены основные статистики: среднее, стандартное отклонение, минимум, максимум, квартили.
4. Исследование категориальных признаков

Для ключевых категориальных признаков (Item Purchased, Category, Location, Size, Color, Payment Method, Frequency of Purchases) выведены уникальные значения и их частоты.
5. Анализ корреляций

Построена корреляционная матрица для числовых признаков — значимых линейных зависимостей не обнаружено.
6. Визуализация

Построена гистограмма с кривой распределения для признака Previous Purchases (количество предыдущих покупок).
